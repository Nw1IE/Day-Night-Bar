# «День / Ночь» — Элитный бар

Репозиторий содержит фронтенд-часть веб-сайта для элитного бара «День / Ночь». Проект представляет собой современный лендинг с интерактивными элементами, системой бронирования и административной панелью для управления контентом.

-----

## Предназначение

Сайт разработан как визитная карточка заведения премиум-класса. Основные цели проекта:

  * **Визуальная презентация:** Демонстрация атмосферы бара и авторских коктейлей через анимированные элементы.
  * **Информирование:** Удобный просмотр актуального меню и действующих акций.
  * **Конверсия:** Упрощение процесса бронирования столиков для клиентов.
  * **Управление:** Оперативное обновление объявлений и позиций меню через встроенную админ-панель.

-----

## Технологический стек

Проект выполнен с использованием чистого стека (Vanilla), что обеспечивает высокую скорость загрузки и отсутствие лишних зависимостей:

  * **HTML5:** Семантическая разметка, использование современных тегов для SEO.
  * **CSS3:** 
      * Custom Properties для легкой настройки цветовой схемы.
      * Flexbox и Grid для адаптивной верстки.
      * Keyframes для плавных анимаций слайдера и интерактивных элементов.
  * **JavaScript (ES6+):** 
      * Модульный подход.
      * DOM Manipulation для управления состоянием корзины, меню и админки.
      * Обработка событий клавиатуры и навигации.
  * **Font Awesome:** Библиотека иконок для интерфейса.

-----

## Архитектура проекта

Проект придерживается классической структуры разделения ответственности (**Separation of Concerns**), что делает код читаемым и поддерживаемым.

```text
Day-Night-Bar/
├── css/
│   └── style.css          # Основные стили, переменные и анимации
├── js/
│   ├── main.js            # Инициализация и логика слайдера/меню
│   └── (modules)          # Вспомогательные скрипты управления данными
├── img/                   # Статические изображения и иконки
└── index.html             # Главная точка входа
```

**Ключевые особенности архитектуры:**

1.  **Централизованное управление стилями:** Все ключевые цвета (золотой `--primary`, темный `--dark`) вынесены в `:root`.
2.  **Интерактивный слой:** Логика слайдера и переключения вкладок в админ-панели изолирована от разметки.
3.  **Адаптивность:** Использование медиа-запросов для корректного отображения на мобильных устройствах и планшетах.

-----

## Функционал

  * **Анимированный Hero-баннер:** Плавный бесконечный слайдер с изображениями коктейлей.
  * **Система бронирования:** Форма с валидацией данных для резерва столов.
  * **Динамическое меню:** Фильтрация позиций по категориям (коктейли, вино, закуски).
  * **Панель администратора:** 
      * Защищенный вход.
      * Управление списком блюд и акций в реальном времени.
      * Редактирование глобального объявления.

## Запуск проекта

Так как проект написан на чистом стеке без использования сборщиков, запуск максимально прост.

### Через Live Server (Рекомендуется)

Если ты используешь **VS Code**:

1.  Установи расширение `Live Server`.
2.  Нажми правой кнопкой мыши на `index.html`.
3.  Выбери **"Open with Live Server"**.
    *Это обеспечит корректную работу ES-модулей и автоматическую перезагрузку страницы при сохранении кода.*
