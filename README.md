# «День / Ночь» — Элитный бар

Репозиторий содержит как фронтенд, так и бэкенд-часть веб-сайта для бара «День / Ночь». Проект представляет собой современный лендинг с интерактивными элементами и административной панелью для управления контентом.

-----

## Предназначение

Сайт разработан как визитная карточка заведения. Основные цели проекта:

  * **Визуальная презентация:** Демонстрация атмосферы бара и авторских коктейлей через анимированные элементы.
  * **Информирование:** Удобный просмотр актуального меню и действующих акций.
  * **Управление:** Оперативное обновление объявлений и позиций меню через встроенную админ-панель.

-----

## Технологический стек

Проект выполнен с использованием чистого стека, что обеспечивает высокую скорость загрузки и отсутствие лишних зависимостей:

  * **HTML5**
  * **CSS3** 
  * **JavaScript (ES6+)** 
  * **Система сборки Vite**

-----

## Архитектура проекта

Проект придерживается классической структуры разделения ответственности, что делает код читаемым и поддерживаемым.

```text
Day-Night-Bar/
├── css/ 
│   ├── Modules/
│   └── style.css
├── js/
│   ├── Modules/
│   ├── api/
│   └── main.js
├── images/
├── components/
└── index.html
```

-----

## Функционал

  * **Анимированный Hero-баннер:** Плавный бесконечный слайдер с изображениями коктейлей.
  * **Система бронирования:** Форма с валидацией данных для резерва столов.
  * **Динамическое меню:** Фильтрация позиций по категориям (коктейли, вино, закуски).
  * **Панель администратора:** 
      * Управление меню, объявлениями и акциями в реальном времени.

-----

## Запуск проекта

### Через Live Server

Если ты используешь **VS Code**:

1.  Установи расширение `Live Server`.
2.  Нажми правой кнопкой мыши на `index.html`.
3.  Выбери **"Open with Live Server"**.
    *Это обеспечит корректную работу ES-модулей и автоматическую перезагрузку страницы при сохранении кода.*
